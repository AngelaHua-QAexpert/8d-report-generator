## 💡 关键提示

### 必须注意：

1. **API 密钥是必需的**
   - 没有有效的 COZE_LLM_API_KEY，AI 功能无法使用
   - 确保密钥格式正确

2. **部署顺序不能错**
   - 必须先部署后端（Railway）
   - 后端成功后，再部署前端（Vercel）
   - 前端需要后端的地址才能工作

3. **环境变量必须配置**
   - Railway 后端：配置 COZE_LLM_API_KEY
   - Vercel 前端：配置 VITE_API_BASE_URL
   - 配置后必须重新部署
