## 部署步骤

### 第一步：准备环境
- [ ] 注册 Vercel 账号（免费）
- [ ] 注册 Railway 账号（免费）
- [ ] 准备 Git 仓库（可选，可使用 Zip 上传）

### 第二步：部署后端（Railway）
- [ ] 登录 Railway
- [ ] 创建新项目
- [ ] 选择 Deploy from GitHub / Upload ZIP
- [ ] 配置环境变量：
  - `PORT=3000`
  - `NODE_ENV=production`
  - `COZE_API_KEY=你的LLM_API密钥`
- [ ] 等待部署完成（约 5-10 分钟）
- [ ] 获取后端服务 URL

### 第三步：部署前端（Vercel）
- [ ] 登录 Vercel
- [ ] 创建新项目
- [ ] 选择 Deploy from GitHub / Upload ZIP
- [ ] 配置环境变量：
  - `NEXT_PUBLIC_API_URL=你的后端服务URL`
- [ ] 等待部署完成（约 3-5 分钟）
- [ ] 获取前端应用 URL

### 第四步：配置域名（可选）
- [ ] 在 Vercel 配置自定义域名
- [ ] 配置 DNS 记录
- [ ] 验证域名解析
